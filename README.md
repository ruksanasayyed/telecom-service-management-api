# Telecom Service Management API
## Overview

Telecom Service Management API is a backend application built using Java and Spring Boot to simulate enterprise telecom service lifecycle management.

The application manages customers and telecom service orders, supporting operations such as service activation, suspension, cancellation, and status tracking.

This project is being developed to strengthen backend engineering skills and demonstrate enterprise application design.

---

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- REST APIs
- Maven
- Git

---

## Planned Features

### Customer Management
- Create Customer
- Update Customer
- Get Customer Details
- Delete Customer

### Service Order Management
- Create Service Order
- Activate Service
- Suspend Service
- Cancel Service

### Order Tracking
- Track Service Status
- View Order History

- DTO Pattern
